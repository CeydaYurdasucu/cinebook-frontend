Dinamik İçerik Keşfi: API entegrasyonu ile gerçek zamanlı kitap ve film verilerinin listelenmesi ve aranması.

Kişiselleştirilmiş Listeler: Kullanıcıya özel "Okunacaklar", "İzlenecekler" ve "Favoriler" yönetimi.

Modern Arayüz: Kullanıcı dostu, duyarlı (responsive) ve etkileşimli bileşen tasarımı.

Asenkron Veri Yönetimi: REST API'den gelen verilerin verimli bir şekilde yönetilmesi ve UI üzerinde akıcı bir şekilde gösterilmesi.

 Teknik Stack
Frontend: React.js, TypeScript

Styling:  Tailwind CSS 

Veri Yönetimi: Axios / Fetch API

Build Tool: Vite / Create React App
